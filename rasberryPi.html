樹莓派介紹

簡介

「樹梅派」（Raspberry Pi）是一種小型、便宜、功能完整的單板電腦， 由英國的 Raspberry Pi 基金會所開發，最初目的是推廣電腦科學教育， 但因為價格低、體積小、擴充性強，也受到很多開發者、創客（Maker） 和工程師的喜愛。

特色

✅ 體積小：比一張信用卡稍大。 ✅ 價格便宜：基本款價格通常在幾十美元。 ✅ Linux 系統：大多數使用 Raspbian（基於 Debian 的系統）或其他 Linux 發行版。 ✅ 多樣用途：可以拿來做家用伺服器、IoT 裝置、監控系統、機器人、遊戲機模擬器、程式學習平台等。

常見的型號

Raspberry Pi 4：強效處理器、支援 4K 輸出、USB 3.0、可選擇 2~8GB RAM。 Raspberry Pi 5：性能大幅提升，更接近桌機級使用體驗。 Raspberry Pi Zero 系列：體積更小、更省電，適合輕量用途。 Raspberry Pi Pico：基於微控制器，適合硬體控制與嵌入式開發。

可連接的基本外部設備

顯示器（HDMI） 鍵盤滑鼠（USB） 感測器、LED、馬達（透過 GPIO 腳位） 網路（有線或 Wi-Fi）
